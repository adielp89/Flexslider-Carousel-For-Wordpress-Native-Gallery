== Description ==

Wordpress plugin that create an awesome flexslider gallery instead of the default static thumbnails.No custom classes or extra posts necessary, just use the normal add media button and the nice gallery editor already available.


**Latest Update**
Added Setting option under 'media menu setting'.
**Features**
*   Enable plugin from Media Setting under Settings menu.
*   Select image sizes to disply on gallery.
*   No Settings required.
*   Fully responsive gallery.
*   Support almost all themes.


== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.

Note : setting option added under Settings->Media Settings.



== Frequently Asked Questions ==
Can I use my existing WordPress theme?
Yes! Flexslider for Wordpress Native Gallery works out-of-the-box with nearly every WordPress theme.
